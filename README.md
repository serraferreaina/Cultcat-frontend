## Features

- Interface in **Catalan**, **Spanish**, and **English** 🌍
- Navigation with **React Navigation**
- Reusable components with **custom Hooks**
- Synchronization with **REST API**
- User profiles with personal data and preferences

---

## Prerequisites

- 📦 **Node.js ≥ 18**
- ☕ **npm ≥ 9**
- 📱 **Expo Go** (for testing on mobile)
- Project dependencies (`npm install`)

---

## Installation & Running

> Run all commands from the root directory of the project.

### 🐧 Linux/macOS

```bash
# Install dependencies
npm install

# Run the app
npx expo start

# Open with Expo Go or emulator
npx expo run:android
# or
npx expo run:ios
```
### 🪟 Windows

```bash
# Install dependencies
npm install

# Run the app
npx expo start
```

## Project Structure

```
📂
├── __tests__/          # Tests
├── api/                # API configuration
├── app/                # Main components
├── assets/             # Images and visual resources

├── utils/              # Helper functions
├── resources/          # Demo video
├── App.tsx             # Entry point
├── package.json
└── tsconfig.json
```
