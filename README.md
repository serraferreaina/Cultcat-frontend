# 🐱 CultCat -- Projecte d'Enginyeria del Software (PES) 2025

---

## 🚀 Execució local

### 1️⃣ Instal·lació de dependències

```bash
npm install
```

### 2️⃣ Executar en mode desenvolupament

```bash
npx expo start
```

- Obrir amb l'app **Expo Go** al mòbil (QR)

- O bé executar:

  ```bash
  npx expo run:android
  # o
  npx expo run:ios
  ```
